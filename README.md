# 🎬 Movie Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes movie sales data from the IMDb dataset to understand the factors that influence a movie's commercial success. The analysis focuses on revenue, IMDb ratings, runtime, genres, and yearly trends using **Power BI** for interactive visualization.

The objective is to provide actionable insights that can help producers, investors, and marketers make data-driven decisions.

---

## 🎯 Business Objective

- Analyze movie revenue across different genres.
- Study the relationship between IMDb ratings and revenue.
- Examine runtime trends over the years.
- Identify the highest revenue-generating movies.
- Discover patterns that contribute to movie success.

---

## 📂 Dataset

- **Source:** IMDb Movie Dataset
- **Period Covered:** 2006–2016
- **Format:** CSV

### Dataset Features
- Movie Title

- Year
- Runtime
- IMDb Rating

The following preprocessing steps were performed:

2. Removed duplicate records.
3. Handled missing values.
4. Corrected data types.
5. Cleaned and formatted columns.
6. Created calculated measures and columns.
7. Validated data before visualization.

## 📊 Dashboard Features

The dashboard includes:

- 📈 Revenue by Genre
- ⭐ Average IMDb Rating by Genre
- 🎥 Average Runtime Trend (2006–2016)
- 💰 Top 10 Highest Revenue Movies
- 📉 Revenue vs IMDb Rating Analysis
- 📅 Year-wise Revenue, Runtime, and Rating Trends

---

## 🔍 Key Insights

- Action movies generated the highest total revenue.
- Animation and Adventure genres also performed exceptionally well.
- Movies with IMDb ratings between **7 and 8** earned the highest revenues.
- **Star Wars**, **Avatar**, and **Jurassic World** are among the top revenue-generating movies.
- Average movie runtime remained relatively stable between **110–120 minutes**.
- Revenue showed an increasing trend after 2010 while ratings remained consistent.
