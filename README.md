# 🎬 Movie Sales Analysis Dashboard

## 📌 Project Overview

This Power BI project analyzes movie sales data from the IMDb dataset to uncover valuable insights into the factors that influence a movie's commercial success. The dashboard explores revenue trends, IMDb ratings, genres, runtime, and yearly performance to help understand the dynamics of the movie industry.

The dashboard helps answer questions such as:

* Which movie genres generate the highest revenue?
* How do IMDb ratings impact box office performance?
* Which movies are the highest revenue generators?
* How have movie trends changed over the years?
* What factors contribute to a successful movie?

This project converts raw movie data into meaningful business insights through interactive Power BI visualizations.

## 🛠 Tools & Technologies Used

* **Power BI** – Dashboard development and visualization

* **DAX** – Calculated columns and measures

## 🔍 Key Insights

* Action movies generated the highest total revenue.
* Animation and Adventure movies also performed exceptionally well.
* Movies with IMDb ratings between **7 and 8** earned the highest revenue.
* **Star Wars**, **Avatar**, and **Jurassic World** are among the top revenue-generating movies.
* Average movie runtime remained relatively stable over the years.
* Revenue showed an increasing trend after 2010 while ratings remained fairly consistent.
